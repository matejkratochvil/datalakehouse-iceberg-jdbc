Project Overview

This project provides a comprehensive, Docker-based setup to explore the capabilities of Apache Iceberg as an open table format. It integrates various powerful data processing engines and tools, allowing users to experiment with a modern data lakehouse architecture locally. The focus is on ease of setup and hands-on learning, particularly with PyIceberg and a JDBC-based catalog (PostgreSQL).
